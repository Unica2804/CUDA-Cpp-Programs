
## Overview

This project provides optimized machine learning algorithms implemented in C++ with optional CUDA GPU acceleration for high-performance computing tasks.

## Features

- CUDA-accelerated computations
- Efficient C++ implementations
- Support for common ML algorithms

## Requirements

- C++11 or later
- CUDA Toolkit (optional, for GPU acceleration)
- CMake 3.10+
